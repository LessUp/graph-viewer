---
description: Run GraphViewer tests and quality checks
---

1. Run lint:
// turbo
```
npm run lint
```

2. Run all unit tests:
// turbo
```
npm run test
```

3. Run build to catch type errors:
// turbo
```
npm run build
```

4. If the dev server is running, run smoke tests:
```
npm run test:smoke
```

5. Review any failures and fix before committing.
